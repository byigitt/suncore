# Suncore
An open-source version of [nightcore.studio](https://nightcore.studio) - Done due to me want to create some projects.

## Installation

```bash
git clone https://github.com/byigitt/suncore.git
cd suncore
pnpm install
```

## Usage

```bash
pnpm dev
```
